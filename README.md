### 用户登录

师大的学生用户可以使用自己的江西师范大学教务在线的帐号登录该应用

<img src="/Pic/5.jpg" width="180px" height="270px"/>

### 功能菜单

用户登录成功后会进入 APP 的主页面(默认是实时天气页面),功能菜单可以通过点击菜单按钮(上图左上红框内)或者点击手机屏幕从屏幕左边框向右滑动,便可弹出功能菜单,具体功能有实时天气、师大要闻、校园动态、我的课表、课程成绩、考试安排、快递查询、注销登录以及更多设置。

### 实时天气

南昌一年四季天气多变,每天看看天气预报,对一天的行程安排有一定的好处,师大轻览提供了实时天气的功能,并作为该应用主页,让用户一打开 APP 就能看到天气状况,方便快捷,而且在实时天气的功能里,我们还加入了一些小提示,只要点击右下角的扩展菜单就能看到,比如风力向、穿衣建议、运动建议、感冒指数、紫外线强度以及污染程度等让用户更好了解当天天气的情况,为出行做好准备。

### 校园动态

校园动态也是师大新闻网发布关于学生教师工作的最新情况,用户可以通过“校园动态”浏览学校的各种活动,无聊时用来消磨时间还是不错的。

### 我的课表

师大的学生用户可以通过“我的课表”查看自己的当前的课程表,不再为去上课时记不住教室是哪一间而烦恼,不在为上课带错课本而尴尬。

<img src="/Pic/15.jpg" width="180px" height="270px"/>

### 课程成绩

师大的学生用户可以通过“课程成绩”来查看各个学期的期末考试成绩,了解自己的学习状况,以便及时调整学习态度,避免挂科。

<img src="/Pic/16.jpg" width="180px" height="270px"/>

### 考试安排

师大的学生用户可以通过“考试安排”功能查看本学期期末考试安排情况,以便调整复习时间,提高复习效率。

<img src="/Pic/17.jpg" width="180px" height="270px"/>

### 快递查询

用户只要通过点击右下角的黄色的添加按钮,选择快递公司,输入快递单号,确定,便可以查询到快递的最新动态。
